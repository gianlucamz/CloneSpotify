# Clone do Spotify

Este é um projeto do clone do Spotify, desenvolvido para fins educacionais. O objetivo é criar uma interface semelhante à do Spotify, utilizando tecnologias web modernas.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Javascript
- Tailwind CSS (versão 4.x)
- Node.js (versão 24.x)

## Instalação

1. Clone este repositório:

```bash
git clone https://github.com/gianlucamz/CloneSpotify.git
```

2. Navegue até o diretório do projeto:

```bash
cd CloneSpotify
```

3. Instale as dependências:

```bash
npm install
```
